CMSY-168: Developing For The Web


Description:

This course presents in-depth coverage of HTML and cascading style sheets (CSS). Students will learn the core technologies of front-end development - HTML, CSS, and document object model (DOM). The cou`rse also examines recommended practices for creating accessible websites, semantic markup, and emerging technologies. A conceptual overview, design issues, and practical development issues are interwoven. Familiarity and experience with the Windows operating system, the creation of folders (directories) and files, and the World Wide Web is assumed.

Objectives:
  • Design and create web pages which incorporate HTML, CSS, and the standards used for web development.
  • Select, analyze, and choose the appropriate tools for web development, testing, and publishing.
  • Demonstrate the effective use of web development tools.
  • Communicate and collaborate through the use of technology tools.
  • Develop web pages with block, inline, and semantic elements.
  • Create tables, lists, and sections in HTML.
  • Develop web pages with video and audio tags.
  • Discuss accessibility issues, recommendations, technological challenges, and correction steps.
  • Analyze the concepts for creating and using mobile web pages.

CMSY-169: Mobile Design & Development Principles


Description:

The principles and attributes of mobile website and application development are unlike any other approaches on the market and are considerably different from its desktop predecessor. This course provides an overview of the basic principles and theory behind mobile development. Key topics covered will include design functionality, design marketing, phone platforms, and mobile operating systems. Students will obtain a firm understanding of how to create a development strategy, address the mobile context, how to decide which of the multiple mobile development types is best for a customer, and how to create a user experience for it. Students will utilize these core principles to design and implement both a mobile website and basic mobile application.

Objectives:
  • Define mobile ecosystem.
  • Compare mobile to traditional media like print, television, and the Web.
  • Explain unique benefits of the mobile medium.
  • Explain the different types of mobile applications, and their pros and cons.
  • Discuss the different types of context and how mobile devices can address specific contexts.
  • Discuss mobile information architecture, mobile design, future of mobile, and its challenges.
  • Analyze the core technologies of mobile web development.
  • Create a user-centered, context-based mobile strategy.
  • Compare mobile web applications to applications written specifically for particular mobile platform.
  • Utilize techniques for dealing with mobile browser inconsistencies and bugs.
  • Design and develop a mobile web site. Visit a site from a mobile device and run a series of tests.
  • Create iPhone mobile application.
  • Discuss accessibility issues, recommendations, technological challenges, and correction steps.
  • Analyze the concepts of creating and using mobile web.
  • Discuss the new possibilities of mobile web development and its limitations.

CMSY-172: Introduction to Programming with JavaScript


Description:

This course is an introduction to computer programming using JavaScript programming language. It will present the beginning to intermediate topics necessary to create, design, write, test, debug, and document programs for execution in a web browser. This course is designed to teach students how to add interactive capabilities to websites and web applications, how to develop programs utilizing the software development life cycle methodology, and the foundational skills needed for learning other programming languages. It is assumed that students are familiar with HTML and file management.

Objectives:
  • Acquire the tools and problem-solving skills necessary for web site analysis and development.
  • Implement modern web development techniques and standards.
  • Analyze the software development life cycle.
  • Identify JavaScript variables, types of data, functions, and control structures.
  • Add JavaScript to a web site to enable interactivity and enhanced functionality.
  • Execute JavaScript code based on data evaluation, current program state, and browser events.
  • Use object-oriented programming with JavaScript.
  • Navigate and manipulate the web page Document Object Model.
  • Apply current industry standards, conventions, and best programming practices to write efficient code.

CMSY-203: Introduction to PHP


Description:

PHP is an open source server-side scripting language used to create dynamic, data-driven websites for such applications as web-based content management and display systems. It performs many of the same functions as ASP and ColdFusion. In this course students will learn how to use several features of this scripting language. Students will write scripting code within the class and in assignments outside of class.


Objectives:
  • Identify PHP’s capabilities as a server-side scripting language, and the reasons for its popularity.
  • Use a text editor for editing and saving HTML documents and PHP scripts.
  • Manipulate variables and data in PHP.
  • Pass data to PHP documents by using URL parameters and forms.
  • Process strings, numbers, arrays, dates and Boolean values with PHP.
  • Write flow control statements within PHP scripts.
  • Use regular expressions and work with files in PHP.
  • Use PHP to input and output data into a MySQL database to and through a web server.

CMSY-205: Advanced JavaScript

Description:

This course provides comprehensive instruction in JavaScript. It will present both intermediate and advanced topics needed to create, design, write, test, debug, and document programs to run on client machines with JavaScript.


Objectives:
  • Master fundamental concepts regarding the use of JavaScript as a client-side scripting language and the elements of the Document Object Model.
  • Implement looping and conditional branching.
  • Use JavaScript for DHTML, including rollovers and creating pop-up windows.
  • Use JavaScript for form validation, to handle events on a form, and to write scripts for list boxes, check boxes, and radio buttons.
  • Write JavaScript methods to allow web servers to interact with a user.
  • Use JavaScript security tools for password protection.
  • Use JavaScript to write data to cookies and other advanced cookie functions.
  • Develop a troubleshooting strategy to debug common JavaScript errors.
